# Daily Task Start Workflow

작업을 시작하기 전에 에이전트의 역할과 핵심 원칙을 재확인합니다.

1. **역할 정의 로드**: `.agent/rules/01_persona.md` 파일을 읽고 당신의 역할(Quant Developer, Pro Trader, Macro Analyst)을 확인하십시오.
2. **기본 원칙 확인**: `.agent/rules/02_principles.md` 파일을 읽고 언어 정책(한글 필수)과 문서화 규칙을 확인하십시오.
3. **엔지니어링 표준 확인**: `.agent/rules/03_engineering.md` 파일을 읽고 금융 무결성 및 코드 품질 기준을 확인하십시오. (필요 시)

// turbo
4. 준비가 완료되면 현재 작업(Task)을 시작하십시오.
