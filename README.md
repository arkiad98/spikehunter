# SpikeHunter 🏹

> [!IMPORTANT]
> **AI Agent 필독**: 작업 전 `.cursorrules`를 반드시 확인하고 준수하세요.


SpikeHunter는 급등하는 주식을 포착하여 단기 매매 기회를 식별하는 머신러닝 기반 주식 분석 프로젝트입니다.

## 🚀 주요 기능
*   **급등주 패턴 분석**: 과거 데이터를 분석하여 급등 전조 패턴을 학습합니다.
*   **머신러닝 예측**: LightGBM 모델을 사용하여 가격 상승 가능성을 예측합니다.
*   **자동화된 데이터 파이프라인**: 데이터 수집, 전처리, 학습, 예측 과정을 자동화했습니다.
*   **전용 백테스팅**: 다양한 매매 전략을 시뮬레이션하고 검증할 수 있습니다.

## 📂 폴더 구조
*   `config/`: 시스템 설정 및 피처 레지스트리
*   `modules/`: 데이터 수집, 학습, 백테스트 등 핵심 로직 (Core)
*   `addons/`: 분석 및 확장 기능 모듈
*   `dev_space/`: **개발, 디버깅, 분석용 샌드박스** (실험적 코드는 이곳에서 관리)
*   `data/`: 데이터 저장소 (Git 제외 권장)

## 🛠️ 시작하기
이 프로젝트는 Python 기반으로 작성되었습니다.

### 1. 전제 조건
*   Python 3.10 이상
*   `git`

### 2. 설치 방법
코드를 내려받고 필요한 라이브러리를 설치합니다.

```bash
git clone https://github.com/arkiad98/spikehunter.git
cd spikehunter
pip install -r requirement.txt
```

### 3. 데이터 설정 (⚠️ 중요)
이 저장소에는 대용량 주식 데이터 파일이 포함되어 있지 않습니다.
*   **기존 사용자**: 이전 PC의 `data/`, `merged/`, `features/` 폴더를 프로젝트 루트로 복사해오세요.
*   **신규 사용자**: `collect.py` 등을 통해 초기 데이터를 수집해야 합니다 (API 키 필요).

**자세한 설정 방법은 [새 PC 환경 설정 가이드](docs/setup_guide_kr.md)를 참고하세요.**

## 📊 사용법 (파이프라인 실행)
전체 분석 파이프라인을 실행하려면 다음 명령어를 입력하세요.

```bash
python run_pipeline.py
```
실행 후 메뉴에서 원하는 작업(데이터 수집, 전처리, 모델 학습, 예측 등)을 선택할 수 있습니다.
